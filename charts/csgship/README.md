# CSGShip Helm Chart

The **CSGShip** chart deploys the backend service for the IDE coding assistant **CodeSouler**.  
It handles **data shipping** and **task management** across CSGHub services.

## Features

- Backend for CodeSouler IDE
- Data transport between services
- Task management and orchestration

For installation and configuration, refer to the official documentation:  

- [CSGShip Deployment Guide](https://opencsg.com/docs/en/csghub/101/install/kubernetes/csgship)
