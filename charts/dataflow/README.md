# Dataflow Helm Chart

The **Dataflow** chart manages **dataset preprocessing and cleaning**, and integrates with **Label Studio** for labeling tasks.

## Features

- Data preprocessing and transformation
- Data cleaning pipelines
- Integration with Label Studio

For detailed setup and usage, see the official documentation:  

- [Dataflow Deployment Guide](https://opencsg.com/docs/en/csghub/101/install/kubernetes/dataflow)
